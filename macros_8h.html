Detailed Description

Useful C preprocessor macros.

FAQ:

Why these PASTE/CONCAT/JOIN are defined?

These are used to create the symbols (struct names / function names) for different data structure types.

Alternatives:

  • Seperate data structure type code for each type used.
  • void* types, which are converted at runtime;
  • Buffers with a data_size worked with at runtime.
  • Macros expanded with type parameters.
  • Use container_of tricks.
