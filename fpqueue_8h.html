inlinestatic

Copy the values from a source priority queue to a destination priority queue.

Assumes:

  • Source and destination priority queue pointers are not pointing to the same memory.
  • The priority queue pointers are not NULL.
  • The destination priority queue has a capacity that is greater than or equal to source priority queue count.
  • The destination priority queue is an empty priority queue.
Parameters
[in,out]dest_pqueue_ptrThe destination priority queue.
