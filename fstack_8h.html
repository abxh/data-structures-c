Get the value at index.

index starts from the top as 0 and is counted upward to count - 1 as bottom.

Assumes:

  • stack_ptr is not NULL.
  • index is strictly less than stack count.

◆ fstack_copy()

static void fstack_copy ( fstack_type *restrict dest_stack_ptr,
const fstack_type *restrict src_stack_ptr )
inlinestatic

Copy the values from a source stack to a destination stack.

Assumes:

  • Source and destination stack pointers are not pointing to the same memory.
  • The stack pointers are not NULL.
  • The destination stack has a capacity that is greater than or equal to source stack count.
  • The destination stack is an empty stack.
Parameters
[out]dest_stack_ptrThe destination stack.
[in]src_stack_ptrThe source stack.
