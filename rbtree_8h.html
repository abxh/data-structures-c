Detailed Description

Intrusive red-black tree.

Duplicates are not allowed. For supporting duplicate keys, you may store a counter for each node, and count up when a node key already exists.

The memory of the nodes are expected to managed seperately.

  • https://github.com/torvalds/linux/blob/master/include/linux/rbtree.h

Used to compare two keys. This must be manually defined before including this header file.

Is undefined once header is included.

Equality for two keys a and b is defined as !KEY_IS_STRICTLY_LESS(a,b) && !KEY_IS_STRICTLY_LESS(b,a).

Attention
This must be manually defined before including this header file.
Return values
trueIf key a is strictly less than b.
falseIf key a is greater than or equal to b.
